# demosdk
=========

A small library that adds commas to numbers

## Installation

  `npm install @scottatsplyt/demosdk`

## Usage

    var demosdk = require('@scottatsplyt/demosdk');

    var block = demosdk.getBlock();
  
  
  Output should be `35,666`


## Tests

  `npm test`
